# GitHub ENV Exporter

GitHub ENV Exporter is a tool for uplaoding all ENV secrets from .env file to GitHub Actions Secrets.

## Installation

Use the package manager [npm](https://pip.pypa.io/en/stable/) to install foobar.

```bash
npm install
```

## Usage

```bash
node index.js
```

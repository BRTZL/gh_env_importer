# GitHub ENV Importer

GitHub ENV Importer is a tool for importing all ENV secrets from .env file to GitHub Actions Secrets.

## Usage

```bash
npx gh_env_importer --owner=OWNER --repo=REPO --envpath=PATH_TO_ENV
```
